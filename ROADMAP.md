# 🗺️ Roadmap & Future Features

Here is a structured plan for expanding the Event Planner application, broken down into manageable chunks from easy to hard. This serves as a working document to guide future development iterations.

## Phase 1: Core Enhancements (Easy)
- **Search & Filtering:** Allow users to search for events by name, or filter them by date and category.
- **Event Categories/Tags:** Let event creators add tags (e.g., *Tech*, *Social*, *Outdoors*) to make discovery easier.
- **Calendar Integration:** Add an "Add to Calendar" button (Google/Apple) to the event details page.
- **Pagination / Infinite Scroll:** Load events in smaller batches to improve performance as the event list grows.

## Phase 2: Media & Engagement (Medium)
- **Event Cover Photos:** Integrate a cloud storage provider (like UploadThing or Vercel Blob) so organizers can upload custom banner images for their events.
- **Custom User Avatars:** Allow users to upload their own profile pictures instead of relying purely on the Google default.
- **Email Notifications:** Send automated emails for RSVP confirmations or reminders when an event is approaching.

## Phase 3: Advanced Functionality (Hard)
- **Event Comments / Discussion:** Add a live comment section on the event page for attendees to interact and ask questions.
- **Maps Integration:** Display a visual map (e.g., Google Maps, Mapbox) pinpointing the event's location.
- **Ticketing & Payments:** Integrate Stripe to allow organizers to sell tickets for paid events.
